Agent Harness 是什么

Agent Harness 是从模型调用到可治理 Agent 系统的工程化框架，用轻内核插件化架构、运行事实记录和数据库原生数据闭环让 Agent 可持续运行、可排查、可扩展。

AIHarnessAgent工程化可观测性

Agent Orchestration 是什么

Agent Orchestration 是多模型/多Agent协作的编排能力——决定哪个模型做什么、何时做、如何组合，它是 AI 时代新的护城河。

AIAgent编排多模型护城河

AI Skill 是什么

AI Skill 是可复用的 Agent 能力封装，内嵌完整 SOP 链路和所需工具，从手动传包正在演进为仓库化标准生态。

AISkill技能封装仓库化

内容出海本地化 是什么

内容出海本地化是 AI 驱动的多语种内容生产管线，通过字幕配音解耦架构和 AI 翻译将新增语种的边际成本归零。

AI内容出海本地化解耦架构边际成本

Context Engineering 是什么

Context Engineering 是设计和管理大模型在多轮交互中应该看到什么信息、何时看到、以什么形式看到的系统工程方法。

Loop Engineering 是什么

Loop Engineering 是围绕 AI Agent 设计可重复运行的反馈系统，让 Agent 能根据目标、上下文、工具结果、验证信号和停止条件持续迭代。

AI 工程化Agent反馈循环工作流

MCP / Model Context Protocol 是什么

MCP 是 Anthropic 提出的模型上下文协议，为 AI 模型连接外部工具和数据源提供标准化接口，本质是一套治理协议，解决"谁有权做什么"。

AIMCP协议工具调用

Prompt Engineer 是什么

Prompt Engineer 通过任务建模、上下文组织、约束设计和评估迭代，让大模型在具体任务中更稳定地产出结果。

RAG / 检索增强生成 是什么

RAG 是让大模型在生成回答前先检索外部知识库的技术，解决模型缺乏私有知识、知识无法实时更新、回答需要可追溯来源的问题。

Spec-Driven Development 是用可执行规格替代传统 PRD 的协作范式——把业务需求转成研发和 AI 工具都能执行的结构化工程上下文。
