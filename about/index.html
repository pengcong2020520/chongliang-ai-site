关于冲量 AI

这是一个 AI 从业者的个人知识资产入口。它不会把内容分散成零碎动态，而是把文章、项目和概念组织成可以长期沉淀的结构。

为什么做这个站

AI 相关内容增长很快，但真正有价值的部分通常不是一条动态，而是一套可以复用的判断框架。这个网站会把你的文章、项目和知识库拆成稳定入口，让读者能够顺着一个概念、一篇文章或一个项目进入你的思考系统。

这个站会如何演进

第一阶段先建立静态 HTML 骨架和双语页面。第二阶段导入 Markdown 文章和 GitHub 项目。第三阶段扩展知识库概念页、内链、结构化数据和 GEO 优化。
